初衷
----------------------------------
在使用 ``Behave`` 框架进行API测试过程中，要先进行登录，获取token,才能调用其它API
项目的登录需要输入验证码，通过 ``pytesser`` 、 ``Tesseract`` 的识别通过率都很低。
在当时的场景是先直接把验证码校验去掉，业余中希望通过 ``libsvm`` 训练可以实现对字符验证码的识别。

----------------------------------
项目功能
----------------------------------
* 生成验证码
* 使用识别模型预测未知验证码图片
