构建项目运行环境
===============

项目依赖库
----------
* \ `pipenv==11.9.0 <https://docs.pipenv.org/>`_
* `pybuilder==0.11.12 <http://pybuilder.readthedocs.io/en/latest/>`_

构建env环境
-----------
* 操作步骤
.. code-block:: bash

    git clone git@github.com:seekplum/identify-captcha.git
    cd identify-captcha
    pyb
    source .venv/bin/activate
    python src/train_data.py
