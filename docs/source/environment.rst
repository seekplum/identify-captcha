构建项目运行环境
===============

项目依赖库
----------
* \ `pipenv <https://docs.pipenv.org/>`_
* `pybuilder <http://pybuilder.readthedocs.io/en/latest/>`_

构建env环境
-----------
.. code-block:: bash

    git clone git://github.com/seekplum/identify-captcha.git
    cd identify-captcha
    pyb
    source .venv/bin/activate
    python src/train_data.py
